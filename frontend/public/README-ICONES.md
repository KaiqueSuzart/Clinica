# Ícones do PWA

Para gerar os ícones do PWA, você pode usar ferramentas online como:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Ou criar manualmente os seguintes arquivos:

1. **pwa-192x192.png** - Ícone 192x192 pixels
2. **pwa-512x512.png** - Ícone 512x512 pixels  
3. **apple-touch-icon.png** - Ícone 180x180 pixels para iOS
4. **mask-icon.svg** - Ícone SVG para Safari (já criado)

Os ícones devem ter:
- Fundo transparente ou sólido
- Cor principal: #2563eb (azul)
- Design simples e reconhecível
- Boa visibilidade em tamanhos pequenos

Após criar os ícones, substitua os arquivos placeholder na pasta `public/`.
